# mhome-i-resource

## 项目简介

mhome-i-resource 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态 HTML 资源的存储与分发空间。

## 目录结构

```
mhome-i-resource/
├── index.html          # 仓库入口页面（可选）
├── pages/              # 独立 HTML 页面目录
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── assets/             # 公共资源（样式、脚本、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：存放所有独立的 HTML 页面文件，每个文件可自包含所需资源。
- `assets/`：存放页面共享的静态资源文件，便于统一管理和引用。

## 页面归档说明

- 本仓库内的 HTML 页面彼此独立，互不依赖。
- 每个页面可包含完整的 HTML、CSS 和 JavaScript 代码，无需外部服务支持（除 CDN 等通用引用外）。
- 归档页面仅供查阅、演示或参考用途，不保证长期可用性或兼容性。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接打开 `pages/` 目录下的 HTML 文件即可在浏览器中查看。
3. 若需部署，可将整个仓库内容上传至任意静态文件服务器或托管服务。

## 维护说明
